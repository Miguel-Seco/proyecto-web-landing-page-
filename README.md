# Panaderia Doña Rosa

## Descripción
Desarrollo integral de una landing page para un comercio de panificados. El proyecto cuenta con un diseño interactivo estructurado en múltiples secciones (Sobre Nosotros, Productos, Contacto). Implementa funcionalidades como tarjetas de proyectos clickeables, contadores animados de estadísticas y un carrusel bidireccional de clientes.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
